# 🤪 PrankQR

PrankQR is a playful web application that lets users generate QR codes for fun and harmless pranks. By scanning the QR code, users are redirected to unexpected or humorous content, making it perfect for jokes, social experiments, and creative interactions.

🔗 Live Demo: https://prankqr.vercel.app/

---

## ✨ Features

- 🔳 Generate custom QR codes instantly  
- 😂 Redirect to funny or unexpected content  
- ⚡ Fast and lightweight performance  
- 🎨 Simple and user-friendly interface  
- 📱 Mobile-friendly design (easy to scan)  
- 🔗 Shareable QR codes for social use  

---

## 🎯 Purpose

PrankQR aims to:

- Create fun and engaging prank experiences  
- Encourage creativity through QR interactions  
- Provide a simple tool for social entertainment  
- Demonstrate how QR technology can be used playfully  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Libraries (if used):** QR code generator (e.g. qrcode.js)  
- **Deployment:** Vercel  

---

## ⚡ How It Works

1. User enters a target URL (funny video, meme, etc.)  
2. The app generates a QR code linked to that URL  
3. Share or print the QR code  
4. When scanned, it redirects the user to the prank content
